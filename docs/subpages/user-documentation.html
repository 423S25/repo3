User Documentation

How to Login

Go to the site link and locate the button on the right side of the top navigation bar.

Click the button, this should take you to a new login page. Enter a valid username and password to login.

How to Edit a Page

Once logged in and on the home page, find the button (Top Left).

This should bring you into the editing version of the page you were currently looking at.

You can move your cursor around to add blocks (text, images, links, etc.).

In the top right, you can preview in desktop, tablet, or mobile modes.

Top left offers undo/redo, list view, and block addition.

Click the logo (top left) to return to the dashboard to choose and edit other pages.

How to Edit the Template

Once logged in and on the home page, find the button (Top Left).

Clicking it takes you to an editing page with options: Navigation, Styles, Pages, Templates, and Patterns.

If not visible, use the left-hand sidebar to find “Templates”.

Click on “Templates”, then find “Pages” and click it. You’ll now see the template layout (e.g., header, footer).

Add blocks as needed, similar to page editing.

Making an Announcement

Once logged in and on the home page, find the button (Top Left).

This brings you to the Dashboard. Click the relevant tab for announcements.

See a list of previous posts with details (author, category, tags, date).

Edit via “edit (classic editor)” or “block editor” (classic is recommended).

After editing, click “Update” in the top right.

Adding a New Post

Click “Add Post” in the posts section (top left).

Add a title and post body in the editor.

Right sidebar options:

  • Visibility: Choose who can see the post.
  • Publish: Choose when to publish.
  • Tags: Add for organization.
  • Category: Assign for structure.

Click “Publish” when ready.

Adding Categories

Add categories via the “Categories” section in the right-hand taskbar when editing a post.

Adding Tags

Use the “Tags” section in the right-hand taskbar to add or create new tags while editing a post.

Comments

In the dashboard, view post comments by clicking the comment bubbles next to each post.

To manage all comments, use the “Comments” section in the sidebar. Hover to approve or trash comments.

Adding to Calendar

Simply add an event to your synced calendar and it will auto-populate on the site.

Adding a File

Once logged in, find the button (Top Left) to access the Dashboard.

Go to the files tab and click “Add New File”.

Choose a title and upload your file. Optionally, add tags and a description below.

Adding a Link

Navigate to the links page and enter edit mode.

Click into the links block, add a list item, then highlight and use the link button to insert your URL.

Adding an Admin Account

In the Dashboard, navigate to the Users tab and click “Add New User”.

Fill in username, email, and set the role to “Administrator”. Click “Add User” to complete.
