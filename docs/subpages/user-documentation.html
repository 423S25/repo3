User Documentation

Product Description

The HRDC Document and Resource Hub is an internal staff intranet page designed to provide quick access to various resources within the HRDC organization. The platform includes commonly used links and resources, along with a powerful search functionality that enables users to search through HRDC documents. All documents are assignable with tags for easier navigation, and the site can be edited by admin users to keep the content updated and relevant.

How to Run

The HRDC Document and Resource Hub can be accessed directly through your browser. Once logged in, staff members can use the search functionality to find documents, access important links, and update content (if you're an admin user).

How to Use

  • Search: Enter keywords or tags into the search bar to find documents and resources within the system.
  • Access Resources: Click on links and resources listed on the main page to open detailed information.
  • Edit Content (Admin Users Only): Admin users can edit existing documents and update resources through the "Edit" option available next to each item. Remember to save changes after editing.
